Данный файл одержит необходимую информацию о подкаталогах проекта и о программе calculatingOptimalCoins



Информация о подкаталогах проекта:

Подкаталог Documents содержит связанные с разработкой программы документы:
- Внешняя спецификация
- Внутренняя спецификация
- Программа и методика испытаний

Подкаталог Doxygen содержит автоматически сгенерированную документацию программы.

Подкаталог Executable содержит исполняемый файл программы и необходимые зависимости:
- calculatingOptimalCoins.exe (исполняемый файл программы)
- Динамические библиотеки .dll, необходимые для запуска исполняемого файла

Подкаталог calculatingOptimalCoins содержит проект программы и репозиторий

Подкаталог bat_tests содержит функциональные тесты для программы:
- Input (подкаталог с .txt файлами с входными данными для тестов. В нем содержится 20 тестов для проверки ошибочных ситуаций и 20 тестов для проверки нормальной работы программы)
- start.bat (файл для запуска тестов. При запуске он сгенерирует файл test_results.txt с информацией о каждом тесте, а также каталог Output с .txt файлами с результатами работы программы)

Подкаталог tests содержит модульные тесты для программы:
- tests.pro (файл проекта Qt для модульных тестов)
- tst_tests.cpp (файл с модульными тестами для всех функций программы)



Информация о программе calculatingOptimalCoins:

Программа предназначена для расчёта оптимального количества монет каждого номинала для оплаты заданной суммы покупки с использованием алгоритма динамического программирования. Программа находит решение с минимальным количеством монет, а в случае невозможности точной оплаты подбирает вариант с минимальной сдачей.

Программа должна получать два параметра командной строки: имя входного файла с данными о сумме покупки и номиналах монет и имя файла для записи результата.
Пример команды запуска программы: calculatingOptimalCoins.exe C:\\Documents\input.xml .\out.xml.

Для запуска модульных тестов необходимо передать: <путь к программе> --test

Входные данные: файл .txt с XML-разметкой, содержащий тег <config> с вложенными тегами <sum> (сумма покупки, целое число от 1 до 1000) и <nominals> с вложенными тегами <value> (номиналы монет, уникальные целые числа от 1 до 100, не более 100 штук).

Выходные данные: файл .txt с XML-разметкой, содержащий тег <config> с вложенными тегами <change> (сдача) и <nominals> с вложенными тегами <nominal>, каждый из которых содержит теги <value> (номинал) и <count> (количество монет данного номинала).

Программа будет разработана на языке С++ с использованием библиотеки Qt. Дополнительного программного обеспечения не требуется. 
Входные файлы должны быть подготовлены в редакторе Блокнот. Выходные файлы могут быть прочитаны с помощью редактора Блокнот.


Пример работы программы:
input.txt: 
<config>
  <sum>10</sum>
  <nominals>
    <value>2</value>
    <value>5</value>
  </nominals>
</config>

output.txt:
<config>
  <change>0</change>
  <nominals>
    <nominal>
      <value>5</value>
      <count>2</count>
    </nominal>
  </nominals>
</config>
